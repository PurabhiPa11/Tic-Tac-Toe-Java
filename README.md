# Tic Tac Toe (Java GUI)

This project is a simple and interactive Tic Tac Toe game built using Java Swing.  
It provides a graphical user interface where two players can play the game in turns.

## 🧩 Overview
The game consists of a 3×3 grid where players take turns marking cells as X or O.  
The program automatically checks for a winner or a draw after each move.

## ✨ Features
- Two-player gameplay (Player X and Player O)
- Interactive GUI using Java Swing
- Automatic win detection (rows, columns, diagonals)
- Draw detection when board is full
- Restart button to reset the game
- Clean and simple layout

## 🛠️ Technologies Used
- Java
- Swing (for GUI)
- AWT Event Handling
